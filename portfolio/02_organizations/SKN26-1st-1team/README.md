# SKN26-1st-1team Organization

**SK Networks AI Camp 1기 1팀**

SK Networks AI Camp의 첫 번째 팀으로, 데이터 분석 및 탐색적 분석(EDA) 중심의 프로젝트들을 수행했습니다.

## 👥 Team Members
- 장한재, 박민선, 홍완기, 조동휘, 윤찬호, 임승현

## 📋 Projects

### 1. [eda-playground](./01_eda-playground.md)
**Exploratory Data Analysis Training Project**
- **Type**: Educational / Training
- **Language**: Python (Jupyter Notebook)
- **Focus**: 데이터 탐색 및 시각화 기본
- **Key Skills**: 
  - Pandas/NumPy 데이터 조작
  - 데이터 시각화 (Matplotlib, Seaborn)
  - 통계 분석 기초
  - 데이터 정제 및 전처리

### 2. [used_car](./02_used_car.md)
**Used Car Market Data Analysis**
- **Type**: Market Analysis / Business Intelligence
- **Language**: Python
- **Focus**: 중고차 시장 트렌드 분석
- **Key Skills**:
  - 시장 데이터 분석
  - 가격 책정 인사이트
  - 차량 특성 분석
  - 비즈니스 관점의 분석
