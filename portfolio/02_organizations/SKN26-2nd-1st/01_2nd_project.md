# 2nd_project

**SK Networks AI Camp 2기 메인 프로젝트**

## Overview
SK Networks AI Camp 1기의 고급 track인 2기 과정의 메인 프로젝트. 
1차 프로젝트 기간동안 배운 내용에서 습득한 기초 위에 더 복잡한 머신러닝 및 데이터 분석 기법을 적용한 프로젝트입니다.

**Repository**: [SKN26-2nd-1st/2nd_project](https://github.com/SKN26-2nd-1st/2nd_project)

## Team
- 서민혁, 유동현, 윤정연, 전종혁, 정영일, 조동휘

## Project Focus
- 🎓 **Advanced ML**: 고급 머신러닝 기법
- 📊 **Data Science**: 종합적인 데이터 분석
- 🔧 **Feature Engineering**: 특성 공학
- 🎯 **Model Optimization**: 모델 최적화
- 📈 **Performance Metrics**: 성능 평가 및 개선
- 💡 **Practical Application**: 실제 문제 해결

## Technology Stack
- **Primary Language**: Python
- **Libraries**: Pandas, NumPy, Scikit-learn, TensorFlow/PyTorch
- **Format**: Jupyter Notebook
- **Tools**: Data analysis, machine learning, visualization

## Project Phases

### Phase 1: Problem Definition & Data Collection
- 문제 정의 및 목표 설정
- 데이터 수집 및 전처리
- 탐색적 데이터 분석 (EDA)
- 데이터 품질 평가

### Phase 2: Feature Engineering
- 특성 추출 (Feature Extraction)
- 특성 변환 (Feature Transformation)
- 특성 선택 (Feature Selection)
- 차원 축소 기법 (PCA, t-SNE)

### Phase 3: Model Development
- 다양한 머신러닝 모델 실험
- 하이퍼파라미터 튜닝
- 교차 검증 (Cross Validation)
- 앙상블 기법 (Ensemble Methods)

### Phase 4: Model Evaluation & Optimization
- 성능 지표 계산
  - Accuracy, Precision, Recall, F1-Score
  - ROC-AUC, Confusion Matrix
  - RMSE, MAE (회귀 문제)
- 모델 비교 및 분석
- 오버피팅 / 언더피팅 검토

### Phase 5: Deployment & Communication
- 결과 분석 및 해석
- 시각화 및 리포팅
- 주요 인사이트 도출
- 발표 준비

### Data Analysis Methods
- 상관관계 분석 (Correlation Analysis)
- 다중선형회귀분석
- 시계열 분석 (Time Series)
- 이상치 탐지 (Anomaly Detection)

### Optimization Techniques
- 그리드 탐색 (Grid Search)
- 랜덤 탐색 (Random Search)
- Bayesian Optimization
- 조기 종료 (Early Stopping)
