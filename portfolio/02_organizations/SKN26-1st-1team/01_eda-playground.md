# eda-playground

**Exploratory Data Analysis Training Project**

## Overview
SK Networks AI Camp 1기 1팀의 탐색적 데이터 분석 프로젝트. 데이터 분석의 기초부터 시각화까지 종합적인 EDA 학습 자료 및 실습 예제.

**Repository**: [SKN26-1st-1team/eda-playground](https://github.com/SKN26-1st-1team/eda-playground)

## Team
- 장한재, 박민선, 홍완기, 조동휘, 윤찬호, 임승현

## Key Features
- 📊 **Data Exploration**: 데이터 구조 및 특성 파악
- 📈 **Visualization**: Matplotlib, Seaborn을 이용한 시각화
- 🧹 **Data Cleaning**: 결측치 처리, 이상치 탐지
- 📉 **Statistical Analysis**: 기초 통계 분석
- 🔍 **Pattern Recognition**: 데이터 패턴 발견

## Technology Stack
- **Primary Language**: Python (Jupyter Notebook)
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Format**: Interactive Jupyter notebooks
- **Focus**: Data science fundamentals

## Learning Outcomes
- Python 데이터 분석 능력
- 데이터 시각화 기술
- 통계적 사고방식
- 실제 데이터 분석 워크플로우
- 팀 협업 경험

## Skills Demonstrated
- Pandas/NumPy 데이터 조작
- 데이터 시각화
- 통계 분석
- Python 프로그래밍
- Jupyter Notebook 활용
- 팀 프로젝트 관리
