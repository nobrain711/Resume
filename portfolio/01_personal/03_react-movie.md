# react_movie

**Movie Review Platform - React Clone Coding**

## Overview
노마더코드 강좌를 바탕으로 구현한 React 기반 영화 평점 사이트. React 프론트엔드 개발의 기본 개념부터 실제 API 연동까지를 다루는 실습 프로젝트.

**Repository**: [nobrain711/react_movie](https://github.com/nobrain711/react_movie)

## Key Features
- ⚛️ **React Fundamentals**: Components, State, Props
- 🎬 **Movie Database Integration**: 영화 데이터 API 연동
- 🔍 **Search Functionality**: 영화 검색 기능
- ⭐ **Rating System**: 평점 표시 및 관리
- 📱 **Responsive Design**: 모바일 친화적 UI
- 🎯 **User Experience**: 직관적인 사용자 인터페이스

## Technology Stack
- **Primary Language**: JavaScript
- **Framework**: React
- **Styling**: CSS, styled-components
- **API**: The Movie Database (TMDB) API
- **State Management**: React Hooks
- **Focus**: Frontend development, API integration

### 1. Home Page

- 영화 상세 정보 페이지
- 포스터, 줄거리, 출연진
- 사용자 평점 표시

### 4. Responsive Layout
- 모바일 최적화
- 태블릿 대응
- 데스크톱 레이아웃
