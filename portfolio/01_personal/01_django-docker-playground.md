# django-docker-playground

**Django + Docker Development Environment**

## Overview
Docker를 활용한 Django 개발 환경 구축 및 실험 프로젝트. 컨테이너화된 Django 애플리케이션 개발과 배포 프리액티스를 다룹니다.

**Repository**: [nobrain711/django-docker-playground](https://github.com/nobrain711/django-docker-playground)

## Key Features
- 🐳 **Docker Integration**: Django 애플리케이션 컨테이너화
- 🗄️ **Database Setup**: Docker Compose를 통한 DB 연동
- 🚀 **Development Workflow**: 효율적인 개발 환경 구성
- 📦 **Dependency Management**: Python 패키지 관리
- 🔧 **Configuration Management**: 환경별 설정 분리
- 🎯 **Production Ready**: 실제 배포 가능한 구조

## Technology Stack
- **Primary Language**: Python
- **Framework**: Django
- **Containerization**: Docker, Docker Compose
- **Database**: PostgreSQL/MySQL
- **Focus**: DevOps, containerization, deployment

## Key Components

### 1. Docker Setup
- Dockerfile for Django application
- Docker Compose orchestration
- Volume management for development

### 2. Django Configuration
- Settings management (dev/prod)
- Database connection setup

- WSGI/ASGI server configuration

### 3. Development Environment
- Hot reload for code changes
- Database initialization
- Migration management
- Logging setup

### 4. Production Considerations
- Image optimization
- Security best practices
- Database backup
- Health checks
