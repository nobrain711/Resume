**핵심 메시지**: 데이터와 AI로 실제 문제를 해결하는 엔지니어

```
지난 1년간 AI/ML 프로젝트 협업 조직 'Joraemon-s-Secret-Gadgets'에서

프로덕션 시스템을 구축하는 능력을 갖추게 되었습니다.

이 과정에서:
- 복잡한 기술 스택 통합 능력 (Python, TypeScript, SQL, Docker)
- 팀 협업 및 기술 리더십 경험 (7명 팀 프로젝트 2개 주도)
- 데이터 기반 의사결정 능력 (데이터 수집→분석→활용)
```

결과적으로 사용자 만족도 90% 이상을 달성한 프로덕션 시스템을 운영 중입니다.
