# used_car

**Used Car Market Data Analysis Project**

## Overview
중고차 시장 데이터를 분석하는 프로젝트. 
시장 트렌드, 가격대 분석, 인기 차량 정보 등을 데이터 기반으로 파악합니다.

**Repository**: [SKN26-1st-1team/used_car](https://github.com/SKN26-1st-1team/used_car)

## Team
- 장한재, 박민선, 홍완기, 조동휘, 윤찬호, 임승현

## Key Features
- 📊 **Market Analysis**: 중고차 시장 트렌드 분석
- 💰 **Price Analysis**: 차량별 가격대 비교
- 🚗 **Vehicle Classification**: 차량 종류별 분류
- 📈 **Trend Visualization**: 시각화를 통한 인사이트
- 🔍 **Data Exploration**: 탐색적 데이터 분석 (EDA)
- 💡 **Market Insights**: 실용적인 시장 정보 제공

## Technology Stack
- **Primary Language**: Python
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn
- **Format**: Data analysis & visualization
- **Focus**: Market analysis, data insights

## Analysis Areas

### 1. Market Trends
- 시간대별 시장 변화
- 인기 차량 모델 추이
- 가격 변동 패턴
- 거래량 분석

### 2. Price Analysis
- 차량 연식별 가격 변화
- 브랜드별 가격 비교
- 차급별 가격대
- 옵션별 가격 프리미엄

### 3. Vehicle Characteristics
- 연비 분석
- 배기량 분포
- 색상 선호도
- 변속기 타입 분포

### 4. Statistical Insights
- 상관관계 분석
- 이상치 탐지
- 분포 분석
- 회귀 분석

## Data Sources
- 중고차 시장 데이터
- 공개 데이터셋
- 웹 크롤링 데이터

## Key Findings
- 🏆 가장 인기 있는 차량 모델

- 📊 연식과 가격의 관계
- 🔄 계절별 거래량 변화
