## 📋 Key Components

### 1. Data Exploration
- 📥 데이터셋 로딩 및 초기 검사
- 📊 기본 통계량 계산
- 🔍 데이터 분포 분석
- ⚠️ 이상치 탐지

### 2. Data Visualization
- 📉 단변량(Univariate) 시각화
  - Histogram, KDE plot, Box plot
- 📊 이변량(Bivariate) 시각화
  - Scatter plot, Heatmap, Pair plot
- 🎨 다변량(Multivariate) 시각화
  - Correlation matrix, 3D plots
- 🖥️ 인터랙티브 대시보드

### 3. Data Preprocessing
- 🔧 결측치(Missing values) 처리
- 🎯 이상치(Outliers) 처리
- 📏 정규화/표준화(Normalization/Standardization)
- ⚙️ 특성 엔지니어링(Feature Engineering)

### 4. Statistical Analysis
- 📊 기술통계(Descriptive Statistics)
- 📈 가설 검정(Hypothesis Testing)
- 🔗 상관분석(Correlation Analysis)
- 📉 회귀분석(Regression Analysis)
