# olfit

**AI-Powered Perfume Recommendation System**

## Overview
6명의 팀이 함께 만든 향수 추천 시스템. AI 기반 향수 추천 엔진, 데이터 수집, 처리, 및 웹 어플리케이션을 포함한 엔드-투-엔드 프로젝트.

**Repository**: [Joraemon-s-Secret-Gadgets/olfit](https://github.com/Joraemon-s-Secret-Gadgets/olfit)

## Team
이창우, 장한재, 전승권, 전종혁, 조동휘, 최수아, 홍완기

## Key Features
- 🌸 **Perfume Recommendation Engine**: AI-powered personalized recommendations
- 🔍 **Advanced Search**: Similarity-based search with embeddings
- 📊 **Data Processing**: Multi-source data aggregation and cleaning
- 🎨 **Modern UI/UX**: Responsive web interface
- 📈 **RAG-Based Search**: Semantic search with Pinecone embeddings
- 🔄 **5-Axis Aura Matching**: Advanced perfume characteristic matching

## Technology Stack
- **Primary Language**: Python
- **Frontend**: TypeScript/React (4th_react_ts_playground)
- **ML Framework**: PyTorch, embeddings
- **Vector DB**: Pinecone
- **Backend**: FastAPI, Django
- **Data Pipeline**: ETL with data cleaning and enrichment

## Project Components

### 1. Data Layer
- **Crawling**: Web scraping from multiple perfume databases
- **Processing**: Data normalization and enrichment
- **Storage**: Structured data with embeddings

### 2. ML Pipeline
- **Embeddings**: Pinecone-based vector search
- **Recommendation**: Content-based and collaborative filtering
- **Re-ranking**: 5-axis aura similarity scoring

### 3. Application
- **Frontend**: React TypeScript with modern UI
- **Backend**: FastAPI REST API
- **Integration**: Vision analysis for scent detection

### 4. Supporting Services
- Image processing for perfume visuals
- Natural language processing for description
- User profile management

## Use Cases
- **Personal Recommendation**: Find perfect perfume based on preferences
- **Discovery**: Explore similar fragrances
- **Analysis**: Understand perfume characteristics
- **Comparison**: Side-by-side fragrance comparison
- **Shopping Assistant**: Guided purchase decisions
